### Introduction

AudioFFT is an audio spectrum viewing and exporting tool that converts files containing audio streams into high-quality visual spectrograms.

### Features
*   Supports multiple audio stream encoding formats.
*   Provides an interactive spectrogram viewer.
*   Customizable spectrogram generation parameters.
*   Exports spectrograms in various image formats.
*   Batch mode for fully automatic processing of large numbers of files.
*   Provides detailed processing logs.

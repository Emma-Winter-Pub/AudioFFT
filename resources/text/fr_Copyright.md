AudioFFT utilise les ressources tierces suivantes :

---
**Qt 6**
*   Objectif : Interface graphique utilisateur (GUI)
*   Licence : GNU LGPL version 3

**FFmpeg**
*   Objectif : Décodage et lecture audio
*   Licence : GNU LGPL version 2.1 ou ultérieure

**FFTW3**
*   Objectif : Transformée de Fourier rapide (FFT)
*   Licence : GNU GPL version 2 ou ultérieure

**libjpeg-turbo**
*   Objectif : Encodage d’images JPEG
*   Licence : IJG

**libpng**
*   Objectif : Encodage d’images PNG
*   Licence : libpng/zlib

**zlib**
*   Objectif : Compression de données (dépendance de libpng)
*   Licence : zlib

**libtiff**
*   Objectif : Encodage d’images TIFF
*   Licence : style BSD

**OpenJPEG**
*   Objectif : Encodage d’images JPEG 2000
*   Licence : BSD 2-Clause

**libwebp**
*   Objectif : Encodage d’images WebP
*   Licence : BSD 3-Clause

**libavif**
*   Objectif : Encodage d’images AVIF
*   Licence : BSD 2-Clause

**libaom**
*   Objectif : Encodage d’images (dépendance de libavif)
*   Licence : BSD 2-Clause

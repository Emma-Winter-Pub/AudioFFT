### 변경 로그

**V1.1    20260328**

**신규**
*   스트리밍 처리를 추가했습니다.
*   FLAC, ALAC 및 DSD 형식에 대한 멀티스레드 디코딩을 추가했습니다.
*   적응형 32/64비트 부동 소수점 계산 정밀도를 추가했습니다.
*   전체 모드에 대한 동적 메모리 로딩 전략을 추가했습니다.
*   트랙 전환을 추가했습니다.
*   CUE 파일 열기 지원을 추가했습니다.
*   CUE 분할 트랙 전환을 추가했습니다.
*   채널 전환을 추가했습니다.
*   FFT 창 함수 선택을 추가했습니다.
*   스펙트로그램 색상 구성표 선택을 추가했습니다.
*   스펙트로그램 dB 값 조정을 추가했습니다.
*   푸리에 변환 계산 결과 캐싱 메커니즘을 추가했습니다.
*   일괄 처리에 대한 중복 작업 알림을 추가했습니다.
*   지연 보상 기능이 있는 플레이어를 추가했습니다.
*   조정 가능한 십자형 커서를 추가했습니다.
*   전환 가능한 데이터 소스가 있는 프로브를 추가했습니다.
*   주파수 분포 그래프 표시를 추가했습니다.
*   GPU 하드웨어 가속을 추가했습니다.
*   구성 요소 표시/숨기기 제어를 추가했습니다.
*   프레임 속도 조정을 추가했습니다.
*   일괄 처리에 대한 I/O 스케줄링을 추가했습니다.
*   스크린샷 기능을 추가했습니다.
*   설정 패널을 추가했습니다.
*   사용자 구성 저장을 추가했습니다.
*   다국어 지원을 추가했습니다: 간체 중국어, 번체 중국어, 일본어, 한국어, 독일어, 영어, 프랑스어 및 러시아어.
*   높이 값 범위를 확장하고 원본 FFT 포인트 투 포인트 해상도 값을 추가했습니다.
*   시간 정밀도 값 범위를 확장하고 자동 제로 오버랩률을 추가했습니다.
*   매핑 함수 수를 확장했습니다.

**최적화**
*   오디오 디코딩 속도를 최적화했습니다.
*   푸리에 변환 속도를 최적화했습니다.
*   스펙트로그램 렌더링 속도를 최적화했습니다.
*   로그 내용과 레이아웃을 최적화했습니다.
*   스펙트로그램 확대/축소 및 이동의 논리와 부드러움을 최적화했습니다.
*   사용자 인터페이스를 리본 스타일로 변경했습니다.

**수정**
*   APE 형식의 멀티스레드 디코딩 오류를 수정했습니다.
*   일부 파일의 오디오 지속 시간 표시 부정확성을 수정했습니다.
*   FFmpeg 리소스 누수를 수정했습니다.
*   스레드 경합으로 인한 프로그램 충돌을 수정했습니다.
*   일괄 처리 중 푸리에 변환으로 인한 프로그램 충돌을 수정했습니다.
*   이미지 크기가 형식 제한을 초과할 때 일괄 처리의 저장 실패를 수정했습니다.
